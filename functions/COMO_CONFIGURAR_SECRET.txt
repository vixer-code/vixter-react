PASSO A PASSO - Como Configurar Secrets do Stripe

1. Execute o comando:
   firebase functions:secrets:set STRIPE_SECRET_KEY

2. O Firebase vai perguntar:
   "Would you like to use STRIPE_SECRET_KEY=SK_LIVE_... as key instead?"
   
   → Responda: n (não)
   
   Por quê? Porque o Firebase está tentando usar a chave COMO NOME, 
   mas o nome deve ser só "STRIPE_SECRET_KEY"

3. Depois o Firebase vai pedir:
   "Enter a value for STRIPE_SECRET_KEY"
   
   → AÍ SIM você cola a chave completa do Stripe: sk_live_51RHQQB...

4. O Firebase vai confirmar e criar o secret com:
   - Nome: STRIPE_SECRET_KEY
   - Valor: sk_live_51RHQQB... (sua chave do Stripe)

5. Repita para o webhook:
   firebase functions:secrets:set STRIPE_WEBHOOK_SECRET
   → Responda "n" se perguntar algo
   → Cole o valor: whsec_...

RESUMO:
- Nome do secret: STRIPE_SECRET_KEY (sempre esse nome)
- Valor do secret: sk_live_... (sua chave do Stripe quando pedir)
